# Processing Prompt — V0

## Usage
After transcribing consultation audio to Japanese text, paste the prompt below into Claude or GPT-4, replacing [TRANSCRIPT] with actual transcript.

## The Prompt

```
あなたは医療コミュニケーションアシスタントです。以下は、日本語で行われた医師と患者の診察の書き起こしです。

あなたの課題：次回の診察前に担当医が確認するための、構造化されたレポートを日本語で作成してください。

レポートの要件：
- 診察中に話された内容に対して臨床的に正確であること（会話の内容を超えた推測はしないこと）
- 適切な医療日本語で記述すること（文書としての敬語、会話中に使用された臨床用語）
- 事実に基づいた観察のみを含めること。臨床的な診断や心理的な評価は含めないこと
- 1〜2ページに収まる簡潔な内容にすること

以下の構成でレポートを作成してください：

──────────────────────────────────────────────
患者診察レポート（AI生成）

診察日：[書き起こしから特定できる場合は記入、不明の場合は「不明」]
患者名：[書き起こしから特定できる場合は記入]
担当医：[書き起こしから特定できる場合は記入]
生成日：[本日の日付]
──────────────────────────────────────────────

## 1. 診察内容の要約
診察で話し合われた内容を2〜3段落で要約。時系列、事実に基づいた三人称の臨床的トーン。

## 2. 主な医療情報
以下の各項目を抽出。会話中に言及されなかった項目は「言及なし」と記載。

### 症状・主訴
### 診断・評価
### 処方・治療
### 検査
### 指示事項

## 3. 次回診察に向けて
会話からの観察。すべて具体的根拠に基づく観察として記述し、臨床的評価や心理学的判断は含めない。

重要：
✅「患者は〜について2回質問した」
✅「患者は〜について追加の説明を求める可能性がある」
❌「患者は不安症の兆候を示している」
❌「患者の健康リテラシーが低い」

## 4. 患者からのメモ
[この欄は患者ご本人が追記します]

──────────────────────────────────────────────
注意事項：本レポートは診察時の録音をもとにAIが生成したものです。
医療記録としての使用を意図したものではありません。
内容の正確性については担当医がご確認ください。

Generated by Kenkoumon — AI-Assisted Patient Visit Report
──────────────────────────────────────────────

書き起こし：
[TRANSCRIPT]
```

## Prompt Tuning Notes
After Experiment Zero, document:
- Transcription issues encountered
- Terms the AI got wrong
- Section 3 quality (too aggressive / too conservative / about right)
- Japanese register feedback from doctor
- Sections doctor found most/least useful
